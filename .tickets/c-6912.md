# known_value_not_in_task_text too strict for read-tool search args

Multiple workspace tasks (UT39, UT8, UT33) fail because MCP tool calls return null mid-session. UT39: all 7 email search strategies return null. UT8: final execute calls return null after connection loss. Model correctly identifies 'MCP connection closed.'
