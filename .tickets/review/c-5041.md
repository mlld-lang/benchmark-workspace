# Typed instruction prereq: rig user-confirmation pause/resume surface

Add a rig confirmation surface for delegated actions that cannot be structurally authorized. The agent should present a specific typed action with resolved targets and tainted payload preview, pause for user confirmation, and resume with a clean, scoped attestation only for the confirmed action.

## Acceptance Criteria

Rig exposes a request_user_confirmation-style operation or equivalent pause/resume flow; confirmed values become clean known attestations scoped to one action id and cannot generalize to other targets; declined or missing confirmation blocks cleanly; UT13 can recover utility through confirmation without allowing autonomous tainted filename-to-control laundering; security tests cover confirmation replay, scope widening, and target swapping.
