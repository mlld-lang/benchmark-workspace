# Shared fast task classifier for deferred primitives

Build a shared fast-model task classifier/router before deferred-area primitives land. It should classify task-level modes such as advice/recommendation, URL promotion/fetch-needed, structured content extraction, and typed-instruction eligibility, returning typed flags the rig can use to select policy bundles and phase constraints. This is the successor/generalization of c-3c7d: the travel deterministic tool router exists, but the deferred security primitives need a shared task classifier rather than per-primitive ad hoc classification.

## Acceptance Criteria

Classifier has zero-LLM stubs and live fast-model path; outputs typed flags with confidence/reason; advice flag gates no-influenced-advice policy bundle; URL/structured-extract/typed-instruction designs can consume the same classifier output; targeted tests cover false positive benign tasks and false negative attack slices.
